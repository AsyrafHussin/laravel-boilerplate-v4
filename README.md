# Laravel Boilerplate

My Laravel Boilerplate v4

## Laravel Version

9.34.0

## Getting Started

1.Clone the repo

```bash
git clone git@github.com:AsyrafHussin/laravel-boilerplate-v4.git <project-name>
```

2.Navigate to clonned folder

```bash
cd <project-name>
```

3.Run initial setup

```bash
./setup.sh
```

4.Serve the application

```bash
./serve
```
